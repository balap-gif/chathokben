📌 Langkah 5: Ganti firebaseConfig di bawah dengan data Anda

📌 Langkah 6: Atur Rules Database (Realtime Database → Rules):

{
  "rules": {
    ".read": "auth != null",
    ".write": "auth != null"
  }
}
